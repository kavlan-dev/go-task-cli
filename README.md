# Go Task CLI

Go Task CLI - это простое консольное приложение для управления задачами, написанное на языке Go. Оно позволяет пользователям создавать, обновлять, удалять и отслеживать задачи с различными статусами.

## Функции

- **Добавление задач**: Пользователи могут добавлять новые задачи с описанием.
- **Обновление задач**: Пользователи могут обновлять описание существующих задач.
- **Удаление задач**: Пользователи могут удалять задачи по их идентификатору.
- **Отметка статуса задач**: Пользователи могут отмечать задачи как "в процессе" или "выполнено".
- **Просмотр задач**: Пользователи могут просматривать все задачи или фильтровать их по статусу.

## Установка и запуск

### Предварительные требования

- Установленный Go (версия 1.16 или выше)
- Доступ к командной строке

### Шаги по установке

1. Клонируйте репозиторий:

```bash
git clone https://github.com/kavlan-dev/go-task-cli.git
```

2. Перейдите в директорию проекта:

```bash
cd go-task-cli
```

3. Соберите проект:

```bash
go build -o task-cli
```

4. Запустите приложение:

```bash
./task-cli
```

## Использование

### Добавление задачи

```bash
./task-cli add "Купить молоко"
```

### Обновление задачи

```bash
./task-cli update 1 "Купить молоко и хлеб"
```

### Удаление задачи

```bash
./task-cli delete 1
```

### Отметка задачи как "в процессе"

```bash
./task-cli mark-in-progress 1
```

### Отметка задачи как "выполнено"

```bash
./task-cli mark-done 1
```

### Просмотр всех задач

```bash
./task-cli list
```

### Просмотр задач по статусу

```bash
./task-cli list todo
./task-cli list in-progress
./task-cli list done
```
